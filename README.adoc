== What is Zig?

Zig is a systems programming language that emphasizes:

* **No hidden control flow** - No hidden memory allocations, implicit function calls, or hidden control flow
* **C interoperability** - First-class C/C++ interop without requiring bindings
* **Comptime** - Compile-time code execution and generics
* **No garbage collector** - Manual memory management with safety features
* **Cross-compilation** - Built-in cross-compilation to 50+ platforms

== Prerequisites

=== System Requirements

* Linux (x86_64, aarch64, armv7a, x86), macOS (x86_64, aarch64), or Windows (x86_64)
* curl
* tar with xz support (or xz-utils)

=== Install asdf

[source,bash]
----
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.14.0
echo '. "$HOME/.asdf/asdf.sh"' >> ~/.bashrc
source ~/.bashrc
----

== Installation

=== Add the Plugin

[source,bash]
----
asdf plugin add zig https://github.com/hyperpolymath/asdf-zig-plugin.git
----

=== Install Zig

[source,bash]
----
# List available versions
asdf list all zig

# Install latest stable
asdf install zig latest

# Install specific version
asdf install zig 0.13.0

# Set global version
asdf global zig 0.13.0

# Set local version (per-project)
asdf local zig 0.13.0
----

== Usage

Once installed, use Zig normally:

[source,bash]
----
# Check version
zig version

# Build a project
zig build

# Run a file
zig run hello.zig

# Format code
zig fmt src/

# Cross-compile
zig build -Dtarget=x86_64-linux-gnu
----

=== Environment Variables

|===
| Variable | Description | Default

| `ASDF_ZIG_SKIP_VERIFY`
| Skip checksum verification
| `false`

| `GITHUB_TOKEN`
| GitHub API token for rate limiting
| (none)
|===

== Supported Versions

All official Zig releases from https://ziglang.org/download/ are supported:

* Stable releases (e.g., 0.13.0, 0.12.0)
* Master/nightly builds (use `master`)

== Troubleshooting

=== tar: xz: Cannot exec

Install xz-utils:

[source,bash]
----
# Debian/Ubuntu
sudo apt install xz-utils

# Fedora
sudo dnf install xz

# macOS
brew install xz
----

=== Rate Limiting

If you encounter GitHub rate limiting, set a token:

[source,bash]
----
export GITHUB_TOKEN=your_token_here
asdf install zig latest
----
